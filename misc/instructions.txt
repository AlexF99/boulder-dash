INSTRUCOES:
Pegue os diamantes, cada nivel tem um minimo de diamantes para passar de nivel (indicado no header).
Use as setas do teclado para controlar o rockford.

Use PgUp e PgDn para navegar entre os niveis.
Use a tecla ESC para sair do jogo.
Se morrer, sera apresentada uma tela com as 10 maiores pontuacoes.
Voce pode jogar novamente pressionando a tecla ENTER.
